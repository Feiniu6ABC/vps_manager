### 兼容性确认

代码在 **Ubuntu 22.04 (Python 3.10)** 和 **Ubuntu 24.04 (Python 3.12)** 完全兼容。使用的 `dict | None` 类型注解是 Python 3.10+ 语法，所有标准库模块 (sqlite3, hashlib, subprocess, json, http.server) 都是内置的，**零外部依赖**。

---

### 部署三步走

```bash
# 1. 上传到 VPS
scp -r vpn-manager/ root@VPS_IP:/opt/vpn-manager/

# 2. 安装 sing-box
cd /opt/vpn-manager && python3 main.py --install

# 3. 启动订阅服务器 + 管理面板
vpn-manager --server
# 然后访问 http://VPS_IP:8888/admin
```

### 发卡平台收款流程 (核心回答)

整个链路是这样的：

```
买家 → 发卡网站(独角数卡) → 付款(支付宝/USDT) → 平台回调你的API → 自动创建用户 → 返回订阅链接 → 买家拿到链接
```

具体步骤：

1. **部署发卡平台** - 推荐[独角数卡](https://github.com/assimon/dujiaoka) (开源免费，Docker一键部署)
2. **配置收款** - 在发卡平台后台接入支付宝当面付或 USDT (通过 epusdt)
3. **创建商品** - 每个套餐一个商品，发货方式选「第三方API」，回调地址填 `http://VPS_IP:8888/api/create`，请求体 `{"secret":"你的密钥","plan_id":1}`
4. **买家自助购买** - 买家访问你的发卡网站 → 选套餐 → 付款 → 自动收到订阅链接

推荐 **USDT 收款** (匿名安全)或**支付宝当面付** (国内方便)。详细步骤都写在 `DEPLOY.md` 里了。
