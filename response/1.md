## 新增功能

### 1. Web 管理面板 (`dashboard.py`, 867行)
- **登录认证**: PBKDF2 密码哈希，会话 token，5分钟内最多5次登录尝试
- **概览页**: 活跃用户数、今日/本月/累计营收、带宽/流量利用率仪表盘、CPU/内存/磁盘状态
- **用户管理**: 添加/删除/禁用/续费用户，批量创建并复制订阅链接
- **实时监控**: 每5秒刷新，显示在线用户、连接数、客户端IP、实时上下行流量
- **营收统计**: 按套餐/来源分类，近30天日营收柱状图，近期订单列表
- **操作日志**: 所有管理操作审计记录
- **系统设置**: 套餐编辑(含价格/最大连接数)、服务器容量、API密钥、sing-box管理
- 访问地址: `http://服务器IP:8888/admin`

### 2. VPS 安装器 (`installer.py`, 881行) - 替代 sb.sh
- **SHA256 校验**: 下载 sing-box 后验证哈希
- **无安全漏洞**: 无 `--insecure`、无 sed 注入、使用 `tempfile` 模块、文件权限 0600/0700
- **自动检测**: 系统类型(Debian/Ubuntu/CentOS/Alpine)、架构(amd64/arm64/armv7)、init系统(systemd/openrc)
- **完整流程**: 依赖安装 → BBR优化 → IP检测 → 下载二进制 → 生成证书 → Reality密钥 → 配置文件 → systemd服务
- **升级/回滚**: 升级失败自动回滚到备份

### 3. 你没想到的功能
- **套餐定价与营收追踪**: plans 表新增 `price` 字段，sales 表记录每笔销售
- **操作审计日志**: operation_logs 表记录所有管理操作(谁在什么时候做了什么)
- **最大连接数限制**: 每个套餐可配置 `max_connections`(日卡3/月卡5/高级10)，超出自动踢掉最旧连接，防止订阅分享滥用
- **过期用户自动清理**: 可配置天数(默认7天)后自动删除过期用户
- **系统健康监控**: CPU/内存/磁盘/网络/运行时间/sing-box状态实时显示

### 4. 安全改进(对比 sb.sh)
| 问题 | sb.sh | vpn-manager |
|------|-------|-------------|
| 二进制下载 | 无校验 | SHA256 验证 |
| curl 参数 | `--insecure`, `-k` | 标准 TLS |
| JSON 操作 | sed 注入风险 | 纯 Python |
| 临时文件 | `/tmp/crontab.tmp` | `tempfile` 模块 |
| 文件权限 | `chmod 0666` | `0600`/`0700` |
| 密码存储 | 无 | PBKDF2 + salt |
| 输入验证 | 无 | 域名/端口/UUID 校验 |
