## 一、Cloudflare CDN 备用怎么配置

现在已经内置到系统里了，操作非常简单：

### 前提：你需要一个域名

去 [Namesilo](https://www.namesilo.com) 买一个便宜的，`.top` 域名约 $1/年。注册时打开 WHOIS 隐私保护。

### 步骤

```
第1步: Cloudflare 设置 (网页操作，约5分钟)
├── 注册 Cloudflare 免费账户
├── 添加你的域名
├── 按提示修改域名的 NS 为 Cloudflare 给的 NS
├── 添加 DNS 记录:
│     类型: A
│     名称: vpn  (这样域名就是 vpn.你的域名.top)
│     内容: 你的 VPS IP
│     代理: 打开 (橙色云朵)
└── SSL/TLS → 选择 "Flexible"

第2步: VPS 上配置 (一条命令)
└── vpn-manager → 选 16 → 选 5
    └── 输入你的域名: vpn.你的域名.top
    └── 自动完成: 添加VMess-WS → 重启 → 刷新订阅
```

### 效果

用户的订阅里会自动包含两条线路：
```
节点1: VPN-vl-reality    (直连，速度快，日常使用)
节点2: VPN-CF备用         (走CF CDN，IP被墙时用)
```

V2rayN、Clash 等客户端更新订阅后自动显示两个节点。直连被封了切换到 CF 备用即可。

### CF 优选 IP (提升 CF 线路速度)

CF CDN 默认给的 IP 可能延迟较高。告知用户使用 [CloudflareSpeedTest](https://github.com/XIU2/CloudflareSpeedTest) 工具选出延迟最低的 CF IP，然后在客户端里把 CF 备用节点的地址改为优选 IP。

## 二、怎么用币安账户收 USDT

可以，但不是直接用币安收款。整个链路是这样的：

```
买家付 USDT ──> 你的收款钱包 ──> epusdt 检测到到账 ──> 回调发卡平台 ──> 平台发货
                    │
                    └──> 你手动提到币安 (或自动转)
```

### 具体操作

#### 第1步: 部署 epusdt (开源 USDT 收款网关)

```bash
# 在你的 VPS 上 (可以和 VPN 同一台)
# epusdt 项目: https://github.com/assimon/epusdt

# Docker 部署 (最简单)
mkdir -p /opt/epusdt && cd /opt/epusdt

# 创建配置
cat > .env << 'EOF'
# 你的 TRON (TRC-20) 钱包地址 —— 用来收款的
TRON_ADDRESS=你的TRC20地址
TRON_PRIVATE_KEY=你的私钥

# API
API_AUTH_TOKEN=随便设一个密钥比如abc123

# 汇率 (1 USDT = 多少人民币)
USD_CNY_RATE=7.2
EOF

docker run -d --name epusdt -p 8000:8000 \
  --env-file .env \
  assimon/epusdt:latest
```

#### 第2步: 获取你的 TRC-20 钱包地址

**方法 A: 从币安获取 (最简单)**
1. 打开币安 App → 资金 → 充值 → 搜索 USDT → 选择网络 TRC20
2. 复制那个 `T` 开头的地址
3. 但注意：这是充值地址，不是你控制私钥的钱包

**方法 B: 创建独立钱包 (推荐，更安全)**
1. 下载 TronLink 钱包 (Chrome 扩展或手机 App)
2. 创建新钱包 → 保存好助记词和私钥
3. 这个钱包地址用于 epusdt 收款
4. 收到的 USDT 你可以随时转到币安

**推荐方法 B**，因为：
- 私钥在你手里，更安全
- epusdt 需要私钥来监控交易
- 币安的充值地址不能拿来做这个

#### 第3步: 对接发卡平台

以独角数卡为例：

```
独角数卡后台:
├── 系统设置 → 支付设置 → 添加 epusdt
│   ├── 支付网关地址: http://127.0.0.1:8000
│   └── API Token: abc123 (和 epusdt 配置里的一样)
│
├── 商品管理 → 添加商品:
│   ├── 商品名: "VPN 单日套餐 24h/10GB"
│   ├── 价格: 2.00 (人民币，epusdt 会自动按汇率转成 USDT 金额)
│   ├── 发货方式: 第三方API
│   ├── API地址: http://127.0.0.1:8888/api/create
│   ├── 请求参数: {"secret":"你的vpn-manager密钥","plan_id":1}
│   └── 提取字段: sub_url
│
├── 再创建 "月卡" 商品 (plan_id: 2, 价格 15 元)
└── 再创建 "高级月卡" 商品 (plan_id: 3, 价格 25 元)
```

#### 第4步: 买家购买流程

```
1. 买家访问你的发卡网站 (独角数卡的前端页面)
2. 选择 "VPN 月卡 ¥15"
3. 点击购买 → 选择 USDT 支付
4. 页面显示: "请向 Txxxx...地址 转账 2.08 USDT" (含汇率换算)
5. 买家用任何钱包/交易所转账
6. epusdt 检测到到账 → 通知独角数卡 → 独角数卡调用你的 API
7. 你的 vpn-manager 自动创建用户 → 返回订阅链接
8. 买家在网页上看到: "您的订阅链接: http://1.2.3.4:8888/sub/xxxxx"
```

#### 第5步: 你的提现

```
TronLink 钱包收到 USDT
    ↓
转账到币安 TRC-20 充值地址
    ↓
币安卖出 USDT → 提现到银行卡 (或继续持有)
```

### 整体架构图

```
                        你的 VPS
┌──────────────────────────────────────────────┐
│                                              │
│  sing-box (:443)     ← 用户连接 VPN         │
│       ↑                                      │
│  vpn-manager         ← 管理用户/流量         │
│       ↑                                      │
│  订阅服务器 (:8888)  ← 用户获取订阅          │
│    ├── /sub/token    ← 订阅链接              │
│    ├── /api/create   ← 发卡平台回调 ←──┐     │
│    └── /admin        ← 你的管理面板    │     │
│                                        │     │
│  独角数卡 (:80)      ← 买家下单付款    │     │
│       ↑                    │           │     │
│  epusdt (:8000)      ← USDT到账检测 ──┘     │
│                                              │
└──────────────────────────────────────────────┘
         ↑                        ↑
      买家连 VPN              买家付 USDT
```

三个服务可以跑在同一台 VPS 上，完全够用。
